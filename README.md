# 🚀 Meus Projetos Aleatórios

Bem-vindo ao meu repositório de estudos (studies) e testes! Este espaço é um laboratório pessoal dedicado a pequenos scripts, programas e experimentos que desenvolvo para praticar lógica de programação e explorar novas tecnologias.

**Este repositório é atualizado continuamente conforme eu aprendo novas coisas e conceitos.**

---

## ⚠️ Aviso Importante (Important Disclaimer)

Os códigos contidos neste repositório são **protótipos experimentais** criados exclusivamente para fins de estudo e diversão. 
*   **Sem Foco em Segurança:** Não foram implementadas camadas de segurança (security).
*   **Sem Foco em Melhores Práticas:** O objetivo aqui é testar conceitos e lógica, e não seguir padrões rigorosos de desenvolvimento profissional (best practices).
*   **Uso Pessoal:** Eu não utilizo esses programas no meu dia a dia; eles servem apenas para consolidar meu conhecimento.

**Caso você decida utilizar qualquer código deste repositório, saiba que ele DEVE ser modificado e adaptado para incluir medidas de segurança e boas práticas de programação antes de qualquer uso real.**

## 📂 Projetos no Repositório

### 🎮 Jogos
*   **[Forca](./forca/):** Jogo da forca clássico em C. Utiliza leitura de arquivos externos para carregar as palavras por categorias (animes, plantas, skate, etc.).
*   **[Jogo da Velha (C)](./Jogodavelha_C/):** Implementação do jogo da velha tradicional em C, focado em lógica de matrizes.
*   **[Jogo da Velha (Python)](./Jogodavelha_Py/):** O mesmo jogo, mas em Python, utilizando uma estrutura modular com uma biblioteca própria.
*   **[Snake](./snake/):** O famoso "jogo da cobrinha" feito em C para terminal, utilizando manipulação de entrada de teclado em tempo real.

### 🛠️ Utilitários e Ferramentas
*   **[Lista de Tarefas](./listadetarefa/):** Um gerenciador de tarefas (To-do list) em Python que salva suas atividades em um arquivo JSON.
*   **[Resize Image](./resize/):** Script em Python para redimensionar imagens (JPG) individualmente ou em lote (batch) através de uma lista em arquivo texto.
*   **[Nvim Util](./nvim_util/):** Um pequeno utilitário em Lua para criar janelas flutuantes no Neovim, útil para quem gosta de customizar o editor.

## 🛠️ Tecnologias Exploradas
*   **Linguagens:** C, Python, Lua.
*   **Conceitos:** Manipulação de arquivos, Estruturas de Dados, Persistência JSON, Modularização e Interface de Terminal (CLI).

## 📖 Como usar
Como este é um repositório de scripts isolados, cada pasta contém seu próprio executável ou arquivo principal.
*   **Para C:** Compile usando `gcc nome_do_arquivo.c -o nome_do_executavel`.
*   **Para Python:** Execute com `python nome_do_arquivo.py`.
